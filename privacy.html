Overview

Shop2not is a Chrome extension that helps users extract product data from their Shopify admin dashboard and export it into a CSV format suitable for upload into Whatnot. This policy explains what data is (and isn't) collected, how it's used, and how it complies with the Chrome Web Store Developer Program Policies.

Information Collection and Use

Shop2not does not collect or transmit any personally identifiable information (PII). The extension functions entirely within the user's browser environment and does not send any data to external servers.

  • We do not collect user names, email addresses, or payment information.
  • We do not track browsing history.
  • We do not sell or share any user data.

Permissions Justification

To provide its core functionality, Shop2not uses the following Chrome extension permissions:

  • activeTab – Accesses the currently open Shopify product page to extract product information.
  • host_permissions (https://admin.shopify.com/*) – Required to read product details directly from Shopify's admin dashboard.
  • scripting – Injects a local script to read the DOM of the Shopify page.
  • storage – Stores user-defined settings and temporary session data locally.
  • downloads (if used) – Saves the extracted data as a CSV file on the user's computer.

Data Handling and Storage

  • All data is processed locally in your browser.
  • No data is transmitted, stored, or logged externally.
  • Data is temporarily stored using Chrome's storage.local API and is only accessible to the extension.

Third-Party Services

Shop2not does not use third-party services, analytics tools, or remote code libraries. All code is bundled within the extension.
